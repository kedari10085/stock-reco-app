Privacy Policy - Web Platform

Last Updated: October 22, 2025

1. Information We Collect

Our web platform collects:

  • Account information (email, password)
  • Video call data (for WebRTC functionality)
  • Usage analytics and preferences
  • Investment preferences and portfolio data

2. WebRTC and Video Calls

Video calls are peer-to-peer using WebRTC technology. We store minimal signaling data in Firebase to establish connections. Video/audio streams are not recorded or stored.

3. Firebase Integration

We use Firebase for:

  • User authentication
  • Real-time database for recommendations
  • WebRTC signaling coordination
  • Secure data storage

4. Data Security

All data is encrypted in transit and at rest. WebRTC connections are peer-to-peer encrypted. Firebase provides enterprise-grade security.

5. Contact Us

For privacy questions: privacy@stockreco.com
